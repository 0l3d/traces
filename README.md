# TRACES

Simple syscall tracer program.

## Usage

```bash
git clone https://github.com/0l3d/traces.git
cd traces/
gcc traces.c -o traces
./traces /bin/echo Hello, World!
```

## IMPORTANT

`traces` currently only supports x86_64 linux.

## Features

- Command support more than 200.
- Colored output.
- Register details.
